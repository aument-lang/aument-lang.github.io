The Aument Language

The Aument language is a work-in-progress, dynamically-typed scripting language with

  • performance first: this language is designed with performance in mind. The interpreter is fast and programs can be compiled into even faster native binaries.
  • fast development time: Aument is a dynamic scripting language like Python and Javascript. You can prototype your software then run them directly.
  • fast deployment: once finished, you can compile your code into a native, lightweight executable. You don't even need any external runtime to run it!

class Human {
	var name;
}

def (self: Human) init(name) {
	@name = name;
}

class Cat {}

def (self: Human) say() {
	print "I'm ", @name, "\n";
}

def (self: Cat) say() {
	print "meow!";
}

alice = new Human;
init(alice, "Alice");
bob = new Human;
init(bob, "Bob");
cat = new Cat;

say(alice);
say(bob);
say(cat);
